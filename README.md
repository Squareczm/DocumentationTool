# 🤖 智能文件整理助手 v2.1

一款基于大语言模型的智能文档分类与归档系统，支持**27大类别 + 108个二级分类**的全自动文档分类，真正实现"拖放即归档"的无缝体验。

## ✨ 核心特性

### 🧠 AI驱动的智能分类
- **深度内容理解**：基于大语言模型分析文档内容
- **多层级语义匹配**：主分类 + 二级分类的精准双重定位
- **智能二级文件夹创建**：自动识别文档细分类型，创建合适的子目录
- **多语言支持**：中文、英文、日文无缝处理
- **自学习能力**：自动适应现有文件夹结构

### 📁 27+108全覆盖分类体系
- **企业办公 (15大类 + 60个二级分类)**：
  - 技术开发 → 智能分析/系统架构/开发实践/技术规范
  - 运维管理 → 系统部署/监控告警/安全运维/容器化
  - 项目管理 → 项目规划/需求管理/进度跟踪/团队协作
  - 数据分析 → 业务分析/数据报表/预测建模/市场研究
  - ... 等等
- **个人生活 (12大类 + 48个二级分类)**：
  - 个人财务 → 投资理财/记账管理/保险规划/财务规划
  - 健康医疗 → 医疗记录/健身运动/营养饮食/健康计划
  - 旅行出行 → 旅行计划/游记分享/出行指南/旅行预算
  - 创作写作 → 文学创作/博客文章/创作素材/写作心得
  - ... 等等
- **完整覆盖**：从专业工作到日常生活的全方位精细化分类
- **持续扩展**：支持行业特定分类和自定义扩展

### 🏗️ 智能二级文件夹系统 🆕
```
主分类识别 → 内容细分分析 → 二级分类匹配 → 智能路径生成
```
- **108个精准二级分类**：每个主分类都有4个专业的子分类
- **智能路径生成**：如 `技术开发/智能分析`、`项目管理/需求管理`
- **动态创建策略**：根据文档内容自动选择最合适的二级目录
- **现有文件夹优先**：智能检测并使用已有的文件夹结构

### 🛡️ 四层智能匹配系统
```
精确文本匹配 → 配置化语义匹配 → 相似性匹配 → 智能回退策略
```
确保每个文档都能找到合适的归档位置，**完全避免文件夹爆炸**

### 🔧 企业级配置架构
- **配置化分类规则**：27个主分类 + 108个二级分类的完整配置
- **版本化配置管理**：支持配置版本控制和迁移
- **模板继承系统**：多层次文件夹模板和组织策略
- **性能优化**：缓存、并发处理、批量优化

## 🚀 快速开始

### 📋 前提条件
- ✅ Python 3.8 或更高版本
- ✅ 网络连接（用于API调用）
- ✅ LLM API密钥（OpenAI、DeepSeek等）

### 安装配置

```bash
# 1. 克隆项目
git clone https://github.com/Squareczm/DocumentationToolUpdated
cd DocumentationToolUpdated

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置API密钥
# Windows用户：
copy config.example.yaml config.yaml
# Linux/Mac用户：
# cp config.example.yaml config.yaml
```

**推荐完整配置（以DeepSeek为例）：**
```yaml
llm:
  provider: "openai"
  api_key: "sk-1234567890abcdef..."  # 请替换为您的真实API密钥
  base_url: "https://api.deepseek.com/v1"
  model: "deepseek-chat"
  timeout: 30
```

**获取API密钥：**
- 🔗 **DeepSeek（推荐）**：https://platform.deepseek.com/api_keys

### 🏗️ 配置化分类规则系统

#### 完整的YAML配置架构
```yaml
# config/classification_rules.yaml - 核心配置文件
metadata:
  version: "2.1.0"
  schema_version: "2.0"
  
classification_rules:
  技术开发:
    description: "软件开发、编程、AI技术等相关文档"
    keywords: [AI, 人工智能, 开发, 编程, 架构...]
    target_patterns: [技术方案, AI研究, 开发文档...]
    # 二级文件夹配置
    subfolders:
      智能分析:
        keywords: [AI, 人工智能, 机器学习, 深度学习]
        description: "AI和机器学习相关技术文档"
      系统架构:
        keywords: [架构, 系统设计, 微服务, API]
        description: "系统架构和设计方案"
      # ... 更多二级分类
    priority: 1
```

#### 智能匹配策略
```yaml
# 多重匹配处理
strategy:
  semantic_threshold: 0.02          # 语义匹配阈值
  allow_new_folders: true          # 允许创建新文件夹
  new_folder_strategy:
    prefer_target_patterns: true   # 优先使用目标模式
    high_confidence_threshold: 0.6 # 高置信度阈值
  
  # 二级文件夹策略
  subfolders:
    enabled: true                  # 启用二级分类
    match_threshold: 0.1           # 二级匹配阈值
    max_depth: 2                   # 最大层级深度
```
